# A Classification Benchmark based on the Literary Theme Ontology

This repository contains computer code for reproducing the results numerical described in the manuscript “A Classification Benchmark based on the Literary Theme Ontology.”


## Questions and Feedback
If you have a technical question about the manuscript, feel free to post it as an [issue](https://github.com/theme-ontology/lto-classification-benchmark/issues).

For more open-ended inquiries, we encourage starting a [discussion](https://github.com/theme-ontology/lto-classification-benchmark/discussions).
